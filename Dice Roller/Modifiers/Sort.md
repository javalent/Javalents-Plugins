# Sort

**Syntax**: `XdXsa` / `XdXsd`

Sorts the dice results in ascending or descending order. 

## Examples

| Formula         | Result                   |
| --------------- | ------------------------ |
| `dice: 2d6sa`   | `[2, 3] -> 5`           |
| `dice: 2d6sd` | `[3, 1] -> 4` |
