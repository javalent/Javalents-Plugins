## Drop Lowest

**Syntax**: `XdXdl{n}`

Drops lowest `{n}` rolls. `{n}` is optional, and will default to 1. Dropped dice will display as `Nd`.

### Example

| Formula         | Result                 |
| --------------- | ---------------------- |
| `dice: 2d20dl`  | `[7d, 18] = 18`        |
| `dice: 4d20dl2` | `[4d, 12, 15, 3d = 27` |
